# M2_Temperature_Regulation_System
## Introduction
The system is used to warm the seates of automobiles when used in cold freezing winters.

## Input and connections
* Seat switch (D0)
* Heat switch (D1)
* Heat adjust potentiometer (C0)

## Output
* Heater LED (B0)
* heat-level indicator (B2 & B3)
* Oscilloscope (B1)
